# DesiMart (Vanilla Version)

This is a **plain HTML, CSS, and JavaScript** version of the DesiMart website.
It mimics the innovative React application but requires **no build tools** (like Vite, Webpack, or Node.js).

## How to Run
1. Simply double-click `index.html` to open it in your web browser.
2. Ensure you have an internet connection (it loads Tailwind CSS and Icons from CDN).

## Project Structure
- `index.html`: Contains the layout and markup for all pages (Home, Listing, Auth, Cart).
- `script.js`: Contains all the logic, state management, and mock data.
- `style.css`: (Optional) Minimal custom styles.

## Features Included
- **Dynamic Navigation**: Switch between Home, Shop, and Sign In without reloading.
- **Product Listing**: Search, Filter by Category, and view products.
- **Shopping Cart**: Add items, update quantities, and see "Free Delivery" progress.
- **Mock Services**: Simulates fetching data and placing orders.
- **Support Chat**: A working chat UI with simulated AI responses.
